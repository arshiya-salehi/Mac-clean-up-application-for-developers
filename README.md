# Mac Cleaner Pro

A safer macOS cleanup app built with Electron. It scans supported cache locations first, shows estimated reclaimable storage, lets you choose exactly what to clean, then reports how much was actually removed.

## Run

```bash
npm install
npm start
```

## Package

```bash
npm run build
```

The packaged app is generated by `electron-builder`.

## Cleanup Targets

- User caches: `~/Library/Caches`
- System caches: `/Library/Caches` with an administrator prompt when cleaning
- Homebrew cache and `brew cleanup`
- npm cache
- Xcode DerivedData
- Xcode Archives
- Docker system prune estimate from `docker system df`
- Android and Gradle caches

The app uses a fixed allowlist of cleanup tasks. It does not accept arbitrary paths or shell commands from the UI.
